# Quick Conversion - 便捷数值转换工具 🔄

一个功能强大的在线转换工具，主要用于美元与人民币汇率转换，同时提供多种实用转换功能。专为配置 LiteLLM 的 config.yaml 而设计，未来将持续更新更多转换功能。

## 🌐 在线版本

访问地址：https://usdtocny.netlify.app/

## ✨ 主要功能

### 💱 汇率转换
- **双向转换**：支持美元 ↔ 人民币相互转换
- **实时汇率**：自动获取最新汇率数据，支持多个API源
- **汇率显示**：清晰显示当前汇率和最后更新时间
- **手动刷新**：支持手动刷新汇率数据
- **容错机制**：API失败时自动使用模拟数据

### 🎨 界面特性
- **响应式设计**：完美适配手机、平板和电脑
- **现代化UI**：采用现代设计语言，界面美观简洁
- **主题切换**：支持深色/浅色模式，自动保存用户偏好
- **动画效果**：流畅的交互动画和视觉反馈
- **键盘快捷键**：支持常用快捷键操作

### ⚡ 快速转换工具
- **Token成本计算**：快速计算百万Token的成本（Cost M Tokens → Cost Per Token）
- **存储单位转换**：支持K/M单位到字节的转换（如 8.9K → 9100+ 字节）
- **可折叠面板**：快速转换工具可折叠隐藏，保持界面整洁

## 🛠️ 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **API源**：
  - 主API：exchangerate-api.com
  - 备用API：fxratesapi.com
- **字体**：Google Fonts (Inter)
- **图标**：自定义SVG图标

## 📱 使用说明

### 基本汇率转换
1. 输入要转换的金额
2. 系统自动计算并显示结果
3. 点击"切换方向"按钮可转换货币方向

### 快速转换工具
1. 点击"Quick conversion"展开工具面板
2. **Token成本计算**：输入百万Token成本，自动计算单个Token成本
3. **存储转换**：输入K或M单位的值（如8.9K或6.4M），自动转换为字节数

### 快捷键
- `Enter`：输入完成后确认转换
- `F5` 或 `Ctrl+R`：刷新汇率
- `Shift+Tab`：切换转换方向
